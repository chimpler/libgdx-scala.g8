### Installation

Install [texture-packer-plugin](https://github.com/tlazaro/texture-packer-plugin) as follows:

	git clone https://github.com/chimpler/texture-packer-plugin
	cd texture-packer-plugin
	sbt publishLocal

Run desktop version:

  \$ sbt desktop/run

Run android version:

  \$ sbt android/run
